# **Exercise_Unit5_website**

## **Description**

This repository contains the files for the Unit 5 exercise. The goal of this project is to create a simple website using **Git** and **GitHub**, practicing repository creation, local editing, and synchronization with GitHub.

## **Technologies Used**
- **HTML**
- **CSS**

 ## **How to Open the Website**
1. Download or clone this repository.
2. Locate the file **index.html** in your project folder.
3. Double-click **index.html** or open it in your preferred web browser to view the website.
